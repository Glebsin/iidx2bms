    iidx2bms
</h1>

<div align="center">Tools for converting iidx charts to bms charts</div>

# HOW TO USE

1. Run "setup.bat" if you downloaded zip from "code" tab
2. Put .1 and .s3p or only .ifs files in folder "put .1 and .s3p or only .ifs files here" 
3. If you want put .png and .mp4 with any name in folder "put .png and .mp4 here"
4. Run "run.bat"
5. Follow instructions in console
6. Get result in "result" folder

**Tested on Windows 11, ffmpeg and Python 3.14.2 should be installed and added to PATH, 1 chart per script launch, probably doesn't work with omnimix charts**
